    Dimension reduction

    Principal component analysis

    It is a dimension reduction technique that finds the variance maximizing directions onto which to project the data.

    Eigenvalue, eigenvector Given a matrix A∈Rn×nA\in\mathbb{R}^{n\times n}A∈Rn×n, λ\lambdaλ is said to be an eigenvalue of AAA if there exists a vector z∈Rn\{0}z\in\mathbb{R}^n\backslash\{0\}z∈Rn\{0}, called eigenvector, such that we have:

    Az=λz\boxed{Az=\lambda z}Az=λz​

    Spectral theorem Let A∈Rn×nA\in\mathbb{R}^{n\times n}A∈Rn×n. If AAA is symmetric, then AAA is diagonalizable by a real orthogonal matrix U∈Rn×nU\in\mathbb{R}^{n\times n}U∈Rn×n. By noting Λ=diag(λ1,...,λn)\Lambda=\textrm{diag}(\lambda_1,...,\lambda_n)Λ=diag(λ1​,...,λn​), we have:

    ∃Λ diagonal,A=UΛUT\boxed{\exists\Lambda\textrm{ diagonal},\quad A=U\Lambda U^T}∃Λ diagonal,A=UΛUT​

    Remark: the eigenvector associated with the largest eigenvalue is called principal eigenvector of matrix AAA.


    Algorithm The Principal Component Analysis (PCA) procedure is a dimension reduction technique that projects the data on kkk dimensions by maximizing the variance of the data as follows:

    • Step 1: Normalize the data to have a mean of 0 and standard deviation of 1.
      xj(i)←xj(i)−μjσjwhereμj=1m∑i=1mxj(i)andσj2=1m∑i=1m(xj(i)−μj)2\boxed{x_j^{(i)}\leftarrow\frac{x_j^{(i)}-\mu_j}{\sigma_j}}\quad\textrm{where}\quad\boxed{\mu_j = \frac{1}{m}\sum_{i=1}^mx_j^{(i)}}\quad\textrm{and}\quad\boxed{\sigma_j^2=\frac{1}{m}\sum_{i=1}^m(x_j^{(i)}-\mu_j)^2}xj(i)​←σj​xj(i)​−μj​​​whereμj​=m1​i=1∑m​xj(i)​​andσj2​=m1​i=1∑m​(xj(i)​−μj​)2​
    • Step 2: Compute Σ=1m∑i=1mx(i)x(i)T∈Rn×n\displaystyle\Sigma=\frac{1}{m}\sum_{i=1}^mx^{(i)}{x^{(i)}}^T\in\mathbb{R}^{n\times n}Σ=m1​i=1∑m​x(i)x(i)T∈Rn×n, which is symmetric with real eigenvalues.
    • Step 3: Compute u1,...,uk∈Rnu_1, ..., u_k\in\mathbb{R}^nu1​,...,uk​∈Rn the kkk orthogonal principal eigenvectors of Σ\SigmaΣ, i.e. the orthogonal eigenvectors of the kkk largest eigenvalues.
    • Step 4: Project the data on spanR(u1,...,uk)\textrm{span}_\mathbb{R}(u_1,...,u_k)spanR​(u1​,...,uk​).

    This procedure maximizes the variance among all kkk-dimensional spaces.
